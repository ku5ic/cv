# Sinisa Kusic — CV

Static CV/resume authored in Markdown, built into HTML and PDF.

## Setup

```bash
brew install pandoc
npm install
```

## Build

```bash
npm run build:html   # Markdown → HTML (docs/index.html)
npm run build:pdf    # HTML → PDF (output/cv.pdf)
npm run build        # Both in sequence
```

## Live

Served via GitHub Pages from the `docs/` directory.
